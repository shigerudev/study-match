# Colores — StudyMatch

Paleta base para la interfaz del MVP.

| Uso | Color | Hex |
| --- | --- | --- |
| Fondo principal | Negro profundo | `#0A0A0A` |
| Superficie / tarjetas | Grafito | `#1A1A1A` |
| Superficie elevada | Gris oscuro | `#282828` |
| Texto principal | Blanco | `#F5F5F5` |
| Texto secundario | Gris claro | `#A3A3A3` |
| Acento principal | Violeta StudyMatch | `#8B5CF6` |
| Acento secundario | Azul eléctrico | `#38BDF8` |
| Match / éxito | Verde menta | `#34D399` |
| Guardar / alerta | Amarillo cálido | `#FBBF24` |
| Error / rechazar | Coral | `#FB7185` |

## Uso rápido

- Utilizar el violeta como color principal de llamados a la acción y estados activos.
- Reservar verde menta para matches, confirmaciones y sesiones agendadas.
- Mantener el fondo oscuro para conservar el lenguaje visual inspirado en plataformas de video.
