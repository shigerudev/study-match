# Instrucciones para agentes

Antes de modificar este repositorio, leer y cumplir [`RULES.md`](./RULES.md) y [`SPECS.md`](./SPECS.md).

Usar automáticamente una skill disponible cuando la tarea corresponda a su propósito. Si no hay una skill adecuada disponible, continuar con el mejor flujo seguro y documentar cualquier supuesto relevante.
